# I Seek You

![GitHub](https://img.shields.io/github/license/Lifeni/i-seek-you)

A WebRTC-based cross-platform data transfer application.

## Development

```sh
pnpm i && pnpm run dev
```

If [pnpm](https://pnpm.io/) is not installed, use the `npm i -g pnpm` to install it.
